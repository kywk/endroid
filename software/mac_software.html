[Mac] Favorite Applications 2019

Posted on 2019-03-20 | Comments:

重新檢視與整理所使用的軟體工具, 生活中主要工具幾乎都是網站服務了, 離線軟體大概多以開發工具為主了. 這次整理軟體列表, 同時把可透過 homebrew cask 安裝的工具寫成 script, 往後重灌系統更方便了.

列個清單，當作參考.

開發工具

參考 Mac OS X EI Capitan 10.11.x 開發環境建構

生產力工具

  • Alfred
    必裝神器! 類似 Spotlight 的軟體, 但功能遠遠不止這樣. 能大大減少離開來回鍵盤和觸控板的時間, 詳細教學可以參考這篇教學
  • Magnet
    分割視窗軟體, 支援「拖曳視窗到螢幕邊界後會自動分割」的功能
  • Typora
    基於 Markdown 開源的 IDE
  • XMind
  • Ityscal
    在功能表列上便捷地查看日曆, 可以查看月曆和事件列表, 還能快速創建新事件
  • 截圖
    截圖除了豐富的標註工具, 支援 QR Code 跨螢幕傳圖, 倒數計時截圖, 標籤與貼紙等功能

網路通訊

  • Google Chrome, Chrome Canary, Firefox Developer Edition
  • Franz
    集大成的 IM 通訊軟體, 在同一個視窗內使用 Slack, Messenger, Telegram, …
  • Line
  • Cyberduck
  • Welly 修改版, nally 修改版
  • CloudMounter, rclone
    把网络云盘放进你的Finder里

其他

  • IINA
    必裝神器! Mac 上最強大的播放器, 幾乎支援所有影音格式. 開源, UI 美觀
  • Headset
    極為好用的 Youtube 免費線上音樂播放工具! 豐富歌單, 電台以及個人收藏功能
  • AppCleaner
    刪除肥大的軟體 (如 Adobe) 時，AppCleaner 就是最好解決的方法
  • ExifRenamer
  • McBopomofo 小麥注音輸入法
    小麥注音的隱藏設定
  • BarTender3, Dozer
    有效管理 status bar 出現的 icon, 軟體一多很好用
